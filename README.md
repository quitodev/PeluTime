# PeluTime
<img src="https://i.ibb.co/5Y8KBLD/Grafico.png" width="740">

PeluTime es una aplicación que facilita la reserva de turnos.

Con PeluTime vas a poder reservar turnos en peluquerías, barberías y centros de belleza adheridas a la aplicación.

## ¿Cómo funciona la aplicación?
Una vez que tengas la cuenta creada, podrás seleccionar el lugar que prefieras para luego elegir el horario disponible de cada profesional.

Con PeluTime ya no es necesario comunicarse telefónicamente, sino que simplemente reservás el turno y, una vez confirmado por el local, te llegará una notificación para avisarte que el turno fue reservado.

En caso de que el local no confirme tu turno por diversas razones, también te llegará una notificación de aviso a tu smartphone.

## Beneficios de descargar la aplicación
- Permite descubrir las peluquerías, barberías y centros de belleza más cercanas a tu ubicación actual.
- Podrás reservar un turno de forma práctica y rápida, sin tener que ponerte en contacto telefónicamente con el local.
- También podrás cancelarlos sin inconvenientes, en caso de no poder asistir al turno reservado.
